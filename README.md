Password Fort is a web-based password strength analyzer that helps users evaluate the security of their passwords using entropy calculation, strength scoring, and crack-time estimation. The application provides instant, real-time feedback through a clean and modern user interface.

Features

1. Real-time password strength analysis

2. Strength percentage indicator

3. Entropy calculation (in bits)

4. Estimated password crack time

5. Light & Dark mode support

6. Modern, minimal UI inspired by security tools

7. Privacy-focused (passwords are not stored or transmitted)

Technologies Used

Backend: Python, Flask

Frontend: HTML, CSS, JavaScript

Security Concepts: Password entropy, brute-force attack estimation
